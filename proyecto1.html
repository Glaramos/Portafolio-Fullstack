Descripción

Desarrollé una plataforma e-commerce para la venta online de yerba mate y productos regionales. La aplicación fue diseñada para ofrecer una experiencia simple, rápida y adaptable a distintos dispositivos.

El proyecto permitió digitalizar el proceso de venta, optimizando la navegación, la búsqueda y la gestión de productos dentro de una tienda online funcional.

La solución fue desarrollada con Django, JavaScript, HTML, CSS y Bootstrap, integrando base de datos, administración de productos y diseño responsive enfocado en la experiencia del usuario.

El resultado fue una plataforma moderna, intuitiva y escalable, preparada para mejorar la interacción entre el negocio y sus clientes.
