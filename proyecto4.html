Descripción

Desarrollé una aplicación de escritorio para la gestión de una peluquería canina utilizando Java Swing, JPA y MySQL. El sistema permite realizar operaciones CRUD de manera eficiente, facilitando la carga, visualización y administración de datos mediante una interfaz intuitiva y funcional.
