Descripción

Mini Calculadora es una aplicación web sencilla e interactiva que permite realizar operaciones básicas de suma y resta entre dos números.

Cuenta con una interfaz clara y amigable, diseñada para facilitar la experiencia del usuario.

El proyecto fue desarrollado utilizando Angular, HTML5 y CSS3, aplicando conceptos fundamentales de desarrollo frontend y diseño responsive.
