Descripción

Aplicación de escritorio desarrollada en Java con JPA para la gestión de ventas de automóviles. Implementa un sistema CRUD que permite dar de alta, baja y modificación de registros de vehículos de manera sencilla e intuitiva, utilizando una interfaz gráfica amigable y conexión a base de datos.
